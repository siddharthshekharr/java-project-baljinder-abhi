# E-Commerce Store

A full-featured e-commerce web application built with Java.

## About

This project is an online e-commerce store that provides a complete shopping experience for customers, including product browsing, cart management, and order processing. It is designed with a clean architecture and follows industry-standard practices for building scalable web applications.

- **Baljinder** — Full Stack Developer
- **Sid** — Full Stack Developer

## Tech Stack

- **Language:** Java
- **Build Tool:** Maven / Gradle
- **Backend:** Java Servlets / Spring Boot
- **Database:** MySQL / PostgreSQL
- **Frontend:** JSP / Thymeleaf / HTML, CSS, JavaScript

## Features

- User registration and login
- Product catalog with categories
- Shopping cart functionality
- Order placement and tracking
- Admin panel for product and order management
- Search and filter products
- Responsive UI

## Getting Started

### Prerequisites

- Java 11 or higher
- Maven or Gradle
- A relational database (MySQL / PostgreSQL)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/java-project-baljinder-abhi.git
   ```
2. Navigate to the project directory:
   ```bash
   cd java-project-baljinder-abhi
   ```
3. Configure the database connection in the application properties file.
4. Build and run the project:
   ```bash
   mvn spring-boot:run
   ```
5. Open `http://localhost:8080` in your browser.
